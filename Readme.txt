Temat projektu:

"Zestawienie różnych danych statystycznych przed wybuchem pandemii COVID-19 (przed 03.2020 r.) oraz w czasie jej trwania (po 03.2020 r.), prezentacja zmian wartości w wybranym okresie"

Jak uruchomić:
	uruchomić plik docker-compose.yaml
	włączona aplikacja jest uruchomiona i rozdzielona na kontenery poszczególnych technologii tj. backend, frontend, baza danych


Krótki opis:
	Aplikacja umożliwia logowanie i zabezpieczony dostęp do poszczególnych funkcji backendu.
	Zapewnia export bazy danych jako xml, json i csv
	Zapewnia import pliku json do bazy danych
	Zapewnia import i export do bazy danych
	Wyświetlenie poszczególnych danych z bazy danych przy użyciu tabel i wykresu
